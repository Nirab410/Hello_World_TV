# Hello World TV 

A clean and user-friendly web-based TV streaming app where users can watch live TV channels directly from the browser.

## Live Demo

Visit the app here:  
[helloworldtv.pages.dev](https://helloworldtv.pages.dev/)

## Overview

Hello World TV Web App is designed to provide a simple and smooth live TV watching experience. Users can browse available TV channels and start watching by clicking on a channel card. No extra play button or complicated setup is required.

## Features

- Clean and modern UI
- Responsive design for desktop, tablet, and mobile
- Channel card-based layout
- One-click channel play
- Separate JavaScript file for managing channel data
- Easy to add, update, or remove channels
- Lightweight frontend-only structure
- User-friendly video player section
- Footer with copyright and useful links

## Tech Stack

- HTML
- CSS
- JavaScript
